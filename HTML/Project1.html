What is Programming Languages?

Definition: Programming languages are formal languages comprising a set of instructions that can be used to produce various kinds of output, including software applications, scripts, and algorithms


Types of Programming Languages

  1. High level Languages

  2. Low level Languages

  3. Domain Specific Languages


High Level Languages

High-Level Languages: These are user-friendly and abstracted from machine code, making them easier to read and write. Examples include:

  • Python
  • Java
  • JavaScript
  • Ruby

Low Level Languages

Low-Level Languages: These provide little abstraction from a computer's instruction set architecture. Examples include:

  • Assembly Language
  • C

Display Specific Languages

Domain-Specific Languages (DSLs): Designed for specific tasks or industries. Examples include:

  • SQL
  • HTML
